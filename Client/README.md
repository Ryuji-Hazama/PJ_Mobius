# Project Mobius Client

## Requirements

### `tkinter`

You neek `tkinter` for using `ttkbootstrap`.  
If you have not installed, install it.

```bash
sudo apt update
sudo apt install python3-tk
```

### `PIL.ImageTk`

Also, you need `PIL.ImageTK`.  
Run the following command:

```bash
sudo apt install python3-pil.imagetk
```
